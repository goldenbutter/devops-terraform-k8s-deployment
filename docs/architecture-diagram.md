## Data Flow

```
User Request
     │
     ▼
[Ingress Controller] ──► [ClusterIP Service :80] ──► [Pod :8080]
                                                          │
                                                          ├── GET /         → Health check
                                                          ├── GET /api/data → System info
                                                          ├── GET /api/calc → CPU work (primes)
                                                          └── GET /metrics  → Prometheus metrics
                                                                                   │
                                                                                   ▼
                                                                            [Prometheus :9090]
                                                                              scrape every 15s
                                                                                   │
                                                                                   ▼
                                                                            [Grafana :3000]
                                                                              dashboards + alerts
```

## Autoscaling Flow

```
Load Test (50 concurrent)
     │
     ▼
[/api/calc] ──► CPU spikes above 50%
                      │
                      ▼
               [Metrics Server] ──► [HPA] ──► Scale 2 → 10 pods
                                                │
                                                ▼
                                         [VPA] (Off mode)
                                         Logs recommendations
                                         for right-sizing
```
